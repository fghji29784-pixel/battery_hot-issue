## 2. 측정 원리 — 정전압법 (Potentiostatic Method)

### 2.1 기본 원리

```
[BT2152 연결 구성]

 정밀 전압원 (OCV 매칭)
         │
     µA 전류계
         │
     ┌───┴───┐
     │  셀   │   ← 자가방전 중
     └───────┘

1. 전압원을 셀 OCV에 ±5 µV 이내로 정확히 매칭
2. 셀이 자가방전하면 전압원이 보충 전류 공급
3. 이 보충 전류 = 자가방전 전류(I_SD)
```

**핵심 원리**: 셀이 자가방전하면 전압이 OCV 아래로 내려가려 하지만, 전압원이 이를 막기 위해 전류를 공급한다. 이 전류가 자가방전 전류와 동일하다.

### 2.2 전류 시간 의존성 — 지수형 감쇠 (핵심)

```
측정 시작 직후의 전류 거동:

 전류
  │
  │  A (초기 대전류 – 이완/분극)
  │╲
  │ ╲
  │  ╲
  │   ╲___
  │       ──────── I_SD (정상상태 자가방전 전류)
  │
  └────────────────────── 시간(t)

수식: I(t) = I_SD + A · e^(−t/τ)

여기서:
  I_SD = 진짜 자가방전 전류 (목표값)
  A    = 초기 과잉전류 진폭 (이완·분극 기인)
  τ    = 시정수 = R_series × C_eff
  R_series = 직렬 저항 (전압원 출력 저항 + 배선 저항)
  C_eff    = 셀의 유효 정전용량 (용량/dV)
```

**출처**: Keysight 블로그 "New Resistance Calibration Method..." 2019.10.21, ScienceDirect Energy Reports 2023 초록 확인  
URL: https://www.keysight.com/blogs/tech/bench/2019/10/21/new-resistance-calibration-method-improves-matching-of-li-ion-cell-self-discharge-current-measurement-response-times

### 2.3 시정수(τ)와 측정 시간의 관계

```
τ = R_series × C_eff

측정 시간과 R_series의 관계 (18650 기준):

  R_series = 1–3 Ω   → τ 큼 → 측정 시간: 1–2 시간
  R_series = 0.1–0.2 Ω → τ 감소 → 측정 시간: ~20분
  
  ※ R_series 감소 = 전압원 출력 임피던스 낮춤 + 배선 저항 최소화
```

> **핵심 발견**: "SDM 측정 시간은 출력 저항을 0.1–0.2 Ω으로 설정 시 최소 **20분**까지 단축 가능. 이는 1–3 Ω 대비 시정수를 수배 단축한 것."
> — Energy Reports 2023 논문 (ScienceDirect 초록 기반)

## 4. 관련 핵심 학술 논문

### 4.1 ★★★★★ Sazhin, Dufek, Gering 2017 — CZCP 고속 측정법 (INL/Battelle/DOE)

**논문**: "Enhancing Li-Ion Battery Safety by Early Detection of Nascent Internal Shorts"  
**저자**: Sergiy V. Sazhin, Eric J. Dufek, Kevin L. Gering (Idaho National Laboratory, DOE)  
**게재지**: *Journal of the Electrochemical Society*, 2017, Vol. 164, A6281–A6287  
**DOI**: 10.1149/2.0431701jes  
**URL**: https://www.researchgate.net/publication/311878132

#### 핵심 방법 — CZCP (Current Zero-Crossing Point)

```
[일반 정전압법과 CZCP 비교]

일반법: 전압원 = OCV에 맞춤 → 전류 양(+)에서 감소하여 I_SD에 수렴

CZCP법: 전압원 = OCV보다 약간 낮게 설정 (방전 과전압)
        → 초기 전류 음(−) (셀 방전됨)
        → 전류가 0을 지나는 점 = CZCP
        → 이후 양(+)으로 수렴 = I_SD
```

**왜 CZCP가 빠른가?**
- CZCP 도달 = 명확한 평형점 지시자
- 별도 수렴 판단 불필요
- **측정 시간: 10–20분**

**S-커브 모델 (US20170153290A1과 동일 방법)**:
```
I_TEST(t) = I_POL(1−d·e^(−ct)) − 2(I_POL(1−d))(½ − 1/(1+e^((t/a)^b))) + I_SD,TRUE
```

**적용**: 내부 미세쇼트(Nascent ISC) 조기 검출, 자가방전 전류 기반 안전성 평가  
**의의**: Keysight BT2152와 유사하지만 독립적인 방법론 — INL/DOE가 개발한 공개 알고리즘

### 4.2 ★★★★ Roth et al. 2023 — 이완 효과 분석

**논문**: "Relaxation Effects in Self-Discharge Measurements of Lithium-Ion Batteries"  
**저자**: Roth C. 외 (Technical University of Munich)  
**게재지**: *Journal of the Electrochemical Society*, 2023, 170, 020502  
**DOI**: 10.1149/1945-7111/acb669  
**URL**: https://iopscience.iop.org/article/10.1149/1945-7111/acb669

#### 핵심 내용

**자가방전 측정을 방해하는 이완 효과 2가지**:

| 이완 종류 | 원인 | 지속 시간 | 영향 |
|----------|------|---------|------|
| **단기 전압 이완** | 전극 내 고체상·액상 농도 평준화 | 수 시간 (12–20h) | 측정 초반 교란 |
| **장기 음극 오버행 평준화** | 비활성 음극 영역과 활성 영역 간 Li 이동 | 최대 **2,000시간** | 장기 자가방전 측정 지속 교란 |

**비교된 측정법 3가지**:
- 전압감소법(Voltage Decay)
- 전압유지법(Voltage Hold) — 이 방법이 BT2152 방식
- 용량손실법(Capacity Loss)

**결론**: 장기 아노드 오버행 효과는 표준 국제규격(IEC 62660-1, ISO 12405-4) 28일 테스트에서도 지속됨  
→ **단기(20분) 선별은 장기 아노드 오버행의 영향을 받지 않는다는 전제 필요**

### 4.3 ★★★★ "Fast Method for Calibrated Self-Discharge Measurement" 2023 — 온도 보정 핵심 논문

**논문**: "Fast method for calibrated self-discharge measurement of lithium-ion batteries including temperature effects and comparison to modelling"  
**게재지**: *Energy Reports (Elsevier)*, 2023  
**DOI**: 10.1016/j.egyr.2023.10.xxx (접근 제한, 초록 기준)  
**URL**: https://www.sciencedirect.com/science/article/pii/S2352484723014610

#### 핵심 발견 (초록/검색 결과 기반)

**① 온도 의존성 — Arrhenius 동역학 확인**:
```
자가방전 전류의 온도 의존성:

I_SD(T) = A · exp(−Ea / k_B · T)

여기서:
  Ea = 활성화 에너지 = 0.94 ± 0.14 eV  (NMC 기준, 검증됨)
  k_B = 볼츠만 상수 = 8.617 × 10⁻⁵ eV/K
  T   = 절대온도 (K)
  
  온도별 자가방전 전류 실측값:
  15°C → 10 µA
  40°C → 110 µA
  (약 11배 차이)
```

**② 온도 가속 계수 계산**:
```
T1 = 40°C = 313 K, T2 = 15°C = 288 K

가속 비율 = exp(Ea/k_B × (1/T2 − 1/T1))
          = exp(10,908 × (1/288 − 1/313))
          = exp(10,908 × 0.000277)
          = exp(3.02) ≈ 20.5배

즉: 40°C에서 측정하면 자가방전 신호가 20배 강함
   → 측정 시간을 최대 20배 단축 가능
```

**③ 고속 측정 조합 전략**:
- R_series = 0.1–0.2 Ω 설정 → 시정수 단축
- 지수함수 피팅으로 I_SD 외삽
- → **~20분 측정 가능**

**④ 모델링**: 전기회로 모델 + 3D 온도 유한요소법(FEM) 결합

### 6.2 방법 ①: R_series 최소화 + 지수함수 외삽 (★★★★★ 핵심 방법)

**원리**:
```
I(t) = I_SD + A · e^(−t/τ)
τ = R_series × C_eff

R_series 최소화 → τ 감소 → 10분 데이터로도 I_SD 외삽 가능
```

**실용 파라미터 (Energy Reports 2023 기반)**:
```
R_series = 0.1 Ω 설정 시:
  - τ 수배 단축 (1–3 Ω 대비)
  - 20분 측정으로 I_SD 추출 가능

지수함수 피팅 방법:
  1. I(t) 데이터 수집 (0–10분)
  2. 비선형 최소제곱법(LMA)으로 I_SD, A, τ 추정
  3. I_SD = lim(t→∞) I(t) = 피팅 파라미터

구현 소프트웨어: Python (scipy.optimize.curve_fit), MATLAB, R
```

**주의 사항**:
- R_series 낮을수록 초기 전류 스파이크 커짐 → 증폭기 포화 주의
- 노이즈 대비 신호(SNR) 확인 필요 (I_SD가 매우 작으면 노이즈에 묻힐 수 있음)

---

### 6.3 방법 ②: CZCP (Current Zero-Crossing Point) 방법 (★★★★)

**원리** (Sazhin 2017, INL — US20170153290A1):
```
[CZCP 전류 거동]

 전류
  │
  │
 0─┼──────────────── CZCP ────────────── I_SD (수렴)
  │╲              ↗
  │ ╲            ╱
  │  ╲──────────╱
  │   (음전류 구간: 셀 방전)
  │
  └────────────────────── 시간

방법:
  1. 전압원을 OCV보다 1–10 mV 낮게 설정
  2. 초기에는 전압원이 셀을 방전시킴 → 전류 음(−)
  3. 셀 내부 평형점에서 전류 = 0 (CZCP)
  4. 이후 자가방전 전류만 남아 → I_SD 수렴
  5. CZCP = 명확한 평형 지시자 → 수렴 판단 불필요
```

**측정 시간**: **10–20분** (Sazhin 2017 논문 결과)  
**특허 상태**: US20170153290A1 (Battelle) — 라이선스 필요 시 INL 접촉

---

### 6.4 방법 ③: 온도 가속 + Arrhenius 보정 (★★★★)

**원리**:
```
온도 T_high에서 측정 후 T_ref로 보정:

I_SD(T_ref) = I_SD(T_high) × exp(Ea/k_B × (1/T_ref − 1/T_high))

파라미터 (Energy Reports 2023, JES 2024):
  Ea (NMC) = 0.94 ± 0.14 eV
  Ea/k_B = 10,908 K
  
온도별 가속 계수:
  25°C → 45°C: exp(10908 × (1/298 − 1/318)) = exp(2.3) ≈ 10배
  25°C → 55°C: exp(10908 × (1/298 − 1/328)) = exp(3.7) ≈ 40배
  
예시: 45°C에서 1분 측정 = 25°C에서 10분 측정과 동등한 신호량
```

**구현 방법**:
1. 셀을 45°C 항온조에 넣고 풀충전
2. BT2152B 연결 후 ~2–5분 측정
3. 보정 계수 적용해 25°C 기준 I_SD로 환산

**주의사항**:
- 온도 변화 중 OCV도 변하므로 TCV(온도 계수) 보상 필수
- Keysight 특허 US20180164363A1에 TCV 보상 방법 포함
- 온도 안정화 시간 필요 (보통 30분 이상 항온 유지 후 측정)

---

### 6.5 방법 ④: LG Energy Solution 방식 — 통계 기반 보조 선별

**특허**: US12025673 (LG Energy Solution, Jae Dong Park)

**방법**:
```
1. OCV1 측정 (기준 시점)
2. 5–15분 CV 충전 (50–150 mA) → 좋은 셀 전압 분산 감소
3. OCV2 측정 (15–72시간 후)
4. ΔOCV = OCV1 − OCV2
5. 기준값: 샘플군 ΔOCV 정규분포의 +4σ~+5σ
6. ΔOCV > 기준값 → 불량 판정
```

**의의**: 5–15분 CV 충전으로 정상 셀의 OCV 분산 감소 → 불량 셀 구별력 향상

## 9. 데이터 분석 방법 — 지수형 데이터에서 파라미터 추출

### 9.1 모델 선택

```
BT2152 측정 데이터 모델:

단일 지수: I(t) = I_SD + A · e^(−t/τ)
  - 단순, 빠른 피팅
  - 단기(수십 분) 측정에 적합

이중 지수: I(t) = I_SD + A₁·e^(−t/τ₁) + A₂·e^(−t/τ₂)
  - 빠른 분극(τ₁) + 느린 이완(τ₂) 분리
  - 더 정확, 더 많은 파라미터 필요

Sazhin S-커브: 위 2.4항 참조
  - CZCP법 전용
  - 가장 정확하나 특허 존재
```

### 9.2 피팅 구현 (Python 예시)

```python
import numpy as np
from scipy.optimize import curve_fit

# 모델 함수
def exp_decay(t, I_SD, A, tau):
    return I_SD + A * np.exp(-t / tau)

# 피팅
t_data = ...  # 시간 데이터 (초)
I_data = ...  # 전류 데이터 (µA)

# 초기값 추정
I0_est = I_data[-1]     # I_SD ≈ 마지막 값
A_est  = I_data[0] - I0_est  # A ≈ 초기값 − 최종값
tau_est = 300            # τ ≈ 5분으로 초기 추정

popt, pcov = curve_fit(exp_decay, t_data, I_data,
                       p0=[I0_est, A_est, tau_est],
                       maxfev=5000)

I_SD, A, tau = popt
print(f"자가방전 전류: {I_SD:.2f} µA")
print(f"시정수: {tau:.1f} 초 ({tau/60:.1f} 분)")
```

### 9.3 보정 파이프라인

```
[온도 보정 파이프라인]

측정 온도 T_meas (예: 40°C = 313 K) 에서 I_SD_meas 획득
     ↓
온도 보정 (Arrhenius):
  I_SD_25°C = I_SD_meas × exp(−10908 × (1/298 − 1/313))
            = I_SD_meas × exp(−10908 × −0.000160)
            = I_SD_meas / exp(1.74)
            = I_SD_meas / 5.7
     ↓
판정 기준 비교 (25°C 기준 정상 셀 범위와 비교)
     ↓
OK / NG 판정
```
