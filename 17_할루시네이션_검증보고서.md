## 📋 과방전 이력 없는 셀 — 유효한 정보 요약

**사용자 조건**: 저전압 불량 셀, 과방전 이력 없음 → **경로 A (양극 Cu 이물 기인)**

이 조건에서 관련성 있는 검증된 내용만 요약:

### ✅ 직접 적용 가능 (원문 검증 완료)

**① Cu 이물 → 미세쇼트 발생 메커니즘**
- Cu 용해 전위: **~3.4–3.5V vs Li/Li+** ✅ (다중 문헌 일치)
- 1C 충전 시 Cu 이물 용해 → **2분 내 음극 덴드라이트 핵생성 → 5분 내 미세쇼트** ✅ (Kuwabara 2017 PDF 직접 확인)

**② CV 충전 회복 메커니즘 (Kuwabara 2017)**
- 조건: **3.6V 정전압(CV) 충전** ✅
- 시간: **약 300분** ✅
- 결과: 미세쇼트 완전 해소, 1시간 방치 후 전압 = 정상셀 동등 ✅
- 메커니즘: 국소 저SOC 영역 균등화 → Cu 용해 전위 도달 → 재용해 ✅

**③ Toyota JP 2014-22217 A — 제조 양품화 프로토콜**
- SOC ≤ 20% 방전 선행 ✅
- 재충전 전류: 초기의 **1.5배~5.0배** ✅
- 반복: 최대 **6회** ✅
- 결과: **4/4 불량 전지 전원 양품화** ✅
- 이론: 핵생성 역학, 덴드라이트 돌기 흡수 ✅

**④ Soft ISC 특성**
- 저항 범위: **C/3.7~C/4480** ✅ (J. Energy Storage 2022)
- 가역성: 조건부 가역 (Kuwabara 2017 실증) ✅
- Soft→Hard 전환 시간: 17~25시간 (⚠️ Li metal battery 조건, Li-ion에서는 다를 수 있음)

### ⚠️ 참고 가능하나 주의 필요

**Lee et al. 2024 (고전류 재충전 ISC 악화)**:
- **과방전 이력 없는 셀에는 해당 없음** — 이 경고는 경로 B(과방전 기인) 셀에만 적용
- 사용자 셀(경로 A)에는 Toyota 프로토콜(고전류 재충전) 적용 타당

**Hendricks 2020 (과방전 Cu 비전도성 형태)**:
- **과방전 이력 없는 셀에는 해당 없음** — 경로 B 분석
- 경로 A에서 Cu는 금속 Cu⁰ 상태 (전도성, ISC 형성) ✅

**Li-plating 관련 회복 데이터 (Nature 2024)**:
- **Li 플레이팅 기인 ISC에만 적용** (경로 A Cu 오염과 다름)
- 89%/97% 수치는 Li metal battery 기준, 수치 미검증
- Cu 이물 기인 저전압 셀에는 직접 적용 불가
