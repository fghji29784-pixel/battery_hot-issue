# 06. 머신러닝·딥러닝 기반 배터리 불량 선별 기법 (2023–2026)

---

## 1. 개요

배터리 불량 선별에 머신러닝(ML) 및 딥러닝(DL) 기법을 적용하는 연구가 2023–2026년 급증하였다. Formation 공정 데이터(전압, 전류, 온도, dQ/dV 곡선 등)를 이용한 이상 셀 자동 탐지가 주요 연구 방향이다.

---

## 2. ICA(Incremental Capacity Analysis) / DVA 기반 선별

### 2.1 ICA 개요

- **IC 곡선**: dQ/dV (전압 대비 용량 미분) → 전극 상전이 피크로 배터리 상태 진단
- **DVA**: dV/dQ → 배터리 열화 특성 분석
- Formation 중 CC 충전 데이터로 IC 곡선 자동 생성 → 이상 피크 감지

### 2.2 동적 충전 조건에서 ICA/DVA (2025)

**기존 한계**: ICA/DVA는 정전류(CC) 충전 조건에서만 적용 가능  
**해결책**: 뉴럴네트워크로 동적 충전 조건에서도 ICA/DVA 적용 확장

- **출처**: "Battery State of Health Estimation and ICA under Dynamic Charging Profile Using Neural Networks", *arXiv*, 2025  
  URL: https://arxiv.org/abs/2502.19586

### 2.3 동적 부하 조건 ICA (2025)

- **출처**: "Incremental capacity analysis of battery under dynamic load conditions", *PMC*, 2025  
  URL: https://pmc.ncbi.nlm.nih.gov/articles/PMC12056397/

### 2.4 Mamba 모델 + ICA (SOH 추정, 2025)

- 개선된 고래 최적화 알고리즘(IWO) + Mamba 모델 + ICA 조합
- **출처**: *Ionics*, Springer, 2025  
  URL: https://link.springer.com/article/10.1007/s11581-025-06564-y

## 3. 이상 탐지 알고리즘 벤치마크 ★ 핵심 참고 문헌

**"An Open-Access Benchmark of Statistical and Machine-Learning Anomaly Detection Methods for Battery Applications"**  
URL: https://arxiv.org/html/2511.01745

### 3.1 벤치마크된 15가지 알고리즘

**통계적 방법**:
- SD (Standard Deviation): True Negative Rate ~99.4%, 미세 이상 탐지 한계
- MAD (Median Absolute Deviation): SD와 거의 동일한 성능
- IQR (Interquartile Range)
- Z-score, Modified Z-score

**거리 기반 방법**:
- Euclidean: 원형 경계
- Manhattan: 다이아몬드형 경계
- Mahalanobis: 특성 간 상관관계 처리에 최적

**머신러닝 모델**:
- **iForest (Isolation Forest)**: 일관되게 높은 Recall → 안전 중요 시스템에 최적
- **PCA + Autoencoder**: Recall과 Precision 균형 → 거짓 경보 감소
- KNN (K-Nearest Neighbors)
- GMM (Gaussian Mixture Models)
- Local Outlier Factor

### 3.2 핵심 발견사항

- **배터리 이상탐지 추천**: iForest(높은 Recall) + PCA/Autoencoder(균형) 조합
- 베이지안 최적화로 저성능 모델 크게 개선 가능
- **전이학습(Transfer Learning)**: 다른 배터리 화학계에 걸친 적용에서 기준선 대비 성능 향상

## 5. 머신러닝 모델 유형별 활용법

| 모델 유형 | 입력 데이터 | 출력 | 활성화공정 적용 포인트 |
|---------|----------|-----|------------------|
| Isolation Forest | Formation 전압/전류 시계열 | 이상 셀 점수 | Formation 완료 후 즉시 선별 |
| Autoencoder | IC 곡선, 임피던스 스펙트럼 | 재구성 오차 | 비정상 패턴 자동 감지 |
| CNN | 전압 곡선 이미지화 | 불량 유형 분류 | 다중 결함 유형 동시 분류 |
| GNN | 배터리팩 내 셀 간 관계 | 이상 셀 위치 | 팩 수준 불량 위치 특정 |
| LSTM | 충방전 시계열 | 이상 예측 | Formation 중 조기 경보 |
| SVM/Random Forest | IC 피크 특성값 | 불량 여부 | 해석 가능성 높음 |

## 7. Formation 공정 데이터 기반 선별 파이프라인 (제안)

```
[데이터 수집]
Formation 충방전기 데이터 (V, I, T, 시간)
↓
[전처리]
IC 곡선 생성 (dQ/dV), DVA 곡선 (dV/dQ)
OCV 이완 곡선 추출
↓
[특성 추출]
IC 피크 위치, 높이, 면적
충전 내부저항
OCV 이완 시간 상수
↓
[이상 탐지]
Isolation Forest: 이상 점수 계산
Autoencoder: 재구성 오차 계산
↓
[분류]
이상 점수 > 임계값 → 추가 검사 (EIS, 자가방전)
이상 점수 ≤ 임계값 → 정상 출하
```

## 8. Formation 데이터 기반 수명예측·불량선별 — 2023–2025 핵심 신규 논문 ★

> **공통 아이디어**: Formation 공정 중 이미 수집하는 데이터(전압·전류·용량·OCV·쿨롱효율)를 재활용 → 추가 설비·대기 시간 없이 불량 조기 선별

---

### 8.1 DVA 기반 제조 품질관리 — 인라인 공정 이상 검출 (Frontiers, 2023)

**논문**: "Differential voltage analysis for battery manufacturing process control"  
**저자**: Andrew Weng, Jason B. Siegel, Anna Stefanopoulou (University of Michigan)  
**게재지**: *Frontiers in Energy Research*, 2023  
**DOI**: [10.3389/fenrg.2023.1087269](https://www.frontiersin.org/journals/energy-research/articles/10.3389/fenrg.2023.1087269/full)  
**검증**: WebFetch 직접 확인 ✅

#### 방법 원리

- Formation 완료 후 저전류(C/20~C/30) 방전 곡선에 DVA(dV/dQ) 적용
- 전체 셀 전압 = 양극 OCP + 음극 OCP로 분해 (반전지 참조곡선 피팅)
- 4개 파라미터 추출: 양극 용량(Q₊), 음극 용량(Q₋), 리튬 부정합량(ε), Li 소모량(ΔLi)

#### 검출 가능한 제조 편차

| 지표 | 의미 | 불량 원인 |
|------|------|---------|
| Q₊ 분산 | 양극 로딩 불균일 | 슬러리 코팅 공정 이상 |
| ΔLi (Formation 중 Li 소모량) | SEI 품질 편차 | 전해액/음극 품질 문제 |
| 실용 음-양 비율(N/P ratio) | 리튬 플레이팅 여유도 | 전극 설계·로딩 편차 |

#### 핵심 결과 (제조 사례 연구)

- 동일 화학계 셀 2배치(Mohtat2021 vs. Weng2021)에서 Formation 중 Li 소모량이 **3배 차이** 발견
- 양극 로딩 편차가 음극보다 전체 용량 산포에 더 큰 기여
- 동일 사양서의 셀이라도 제조 연도에 따라 전기화학적 "지문"이 유의미하게 상이

**활성화공정 적용**: Formation 충방전 데이터로 자동 DVA 분석 파이프라인 구축 → 방치 기간 없이 공정 이상 즉시 피드백

### 8.2 딥러닝으로 Formation 데이터에서 리튬 플레이팅 불량 검출 (JEC, 2025)

**논문**: "Deep learning-driven detection of lithium-plating-type defects for battery manufacturing via formation and capacity grading data"  
**저자**: Chaomin Yue 외 (2025)  
**게재지**: *Journal of Energy Chemistry*, Vol. 108, pp. 536–549, 2025  
**URL**: https://www.sciencedirect.com/science/article/abs/pii/S2095495625003407  
**검증**: 제목·저자·권호·페이지 검색 교차 확인 ✅ (paywall, 초록+검색 기반)

#### 핵심 내용

**배경**: 양극 금속이물(금속 오염) → Formation 중 내부단락 유발 → OCV 강하·자가방전 증가  
**문제점**: 기존 방법은 추가 설비나 긴 방치 기간 필요  
**해결책**: **이미 수집 중인** Formation + 용량선별 데이터만으로 DL 모델 학습

**입력 피처**:
- Formation 전압 프로파일
- 초기 쿨롱효율(Initial CE)
- OCV
- 용량
- K-값(자기평준화 효과 지표)

**데이터셋**: 154개 배터리 (정상 106개 + 불량 48개)

**결과**:
- 추가 설비 없이 인라인 불량 검출 가능
- 금속이물 기인 불량 셀은 CE 감소 + OCV/K-값 상승 패턴 → DL이 이 패턴 학습

**⚠️ 검증 주의**: 정확도 수치(%, recall 등)는 원문 paywall로 미확인. 위 내용은 초록·검색 결과 기반

### 8.3 Formation Q(V) 피처 2개로 수명 예측 — 최소 도메인 지식 (Joule, 2025)

**논문**: "Systematic feature design for cycle life prediction of lithium-ion batteries during formation"  
**저자**: Jinwook Rhyu, Joachim Schaeffer, Michael L. Li, Xiao Cui, William C. Chueh, Martin Z. Bazant, Richard D. Braatz  
**소속**: MIT, Stanford, SLAC National Accelerator Laboratory  
**게재지**: *Joule*, March 2025 (arXiv: 2410.07458, 2024)  
**Joule URL**: https://www.cell.com/joule/abstract/S2542-4351(25)00065-0  
**검증**: arXiv HTML 직접 확인 ✅ (저자·결과 일치)

#### 핵심 내용

**목표**: Formation 데이터에서 도메인 지식 최소화로 수명 예측 피처 체계적 설계  
**방법**: 체계적 피처 설계 프레임워크 → Q(V) 곡선에서 최적 전압 구간 자동 탐색

**설계된 피처 (단 2개)**:

| 피처 | 전압 구간 | 의미 |
|------|---------|------|
| Q^B(3.57V) − Q^B(3.60V) | 3.57~3.60V | 마지막 방전 시 특정 전압 구간 용량 차이 |
| Q^B(3.60V) − Q^B(3.66V) | 3.60~3.66V | 동일 방전 중 인접 전압 구간 용량 차이 |

**결과**:
- 수명 예측 **Median 오차 9.20%** (MAPE)
- 최대 오차 11.93%
- autoML 수천 개 모델(155+ 피처)과 동등 성능, **해석 가능성 보유**

**활성화공정 적용**:
- 추가 진단 사이클·추가 대기 없이 Formation 마지막 방전 데이터로 즉시 수명 선별 가능
- 피처 2개의 단순성 → 생산 라인 실시간 계산 부담 최소

### 8.4 SVM 기반 자가방전 전압 강하 예측 (ScienceDirect, 2022)

**논문**: "Self-discharge prediction method for lithium-ion batteries based on improved support vector machine"  
**게재지**: *Journal of Energy Storage*, 2022  
**URL**: https://www.sciencedirect.com/science/article/abs/pii/S2352152X22015596  
**검증**: 제목·저널·URL 검색 확인 ✅ (paywall, 초록 기반)

#### 핵심 내용

- **목표**: 방치 기간 없이 충방전 곡선 피처로 dOCV(자가방전 전압 강하) 예측
- **방법**: 충방전 곡선에서 피처 추출 → Grey Relational Analysis(GRA)로 상관도 높은 피처 3개 선별 → 개선된 SVM 회귀
- **결과**: 3일 방치 없이 자가방전 전압 강하 예측 → 불량 셀 자동 식별

**활성화공정 적용**:
- Formation 충방전기 데이터에서 이미 수집되는 곡선 정보로 dOCV 예측 모델 구축 가능
- 방치 3일을 소프트웨어 예측으로 대체하는 경로
