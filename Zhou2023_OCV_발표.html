배터리 OCV를
6분 안에 예측하는 방법

리튬이온 배터리의 개방회로전압(OCV)을 단 몇 분 만에 정확히 예측하는 신기술 —
활성화공정의 자가방전 선별 시간 단축에 직접 적용 가능

배경 · Problem Statement
왜 OCV 측정이 오래 걸리는가?

배터리에 전류를 흘리면 내부에 분극(Polarization)이 발생합니다. 전류를 끊어도 이 분극이 사라지는 데 시간이 걸리며, 그 시간 동안 전압이 서서히 변합니다. 이것이 이완(Relaxation) 과정입니다.

😫 기존 방법의 문제
⏱️
증분 OCV 테스트: 각 SOC 포인트마다 3시간 대기 × 36 포인트 = 약 115시간
📉
저전류 측정법: 미약하지만 전류가 흘러 엄밀한 OCV가 아님
🧮
곡선 피팅법: 미지수 많고 수식 복잡, 긴 데이터 필요
⚡
능동 전류 인가법: SOC 자체를 변화시켜 오차 발생
✅ 이 논문의 해결책

이완 초기 몇 분의 전압 데이터만으로,
1차/2차 미분비에서 시정수 τ를 실시간 계산 →
완전 이완 없이도 OCV 예측

↑ tc 시점의 미분값만으로 OCV를 계산 — 이완 완료를 기다릴 필요 없음

핵심 이론 · Core Theory
1차 RC 회로 모델과 시정수 τ
전류 차단 후 이완 방정식
전류 = 0 일 때, 분극 전압 Vp는:
Vp(t) = Vp(0) · e-t/τ
단자 전압은:
Vbat(t) = OCV + ΔV₀ · e-t/τ
🎯 핵심 아이디어: τ를 측정값에서 직접 계산

V(t) = OCV + ΔV₀·e-t/τ 를 두 번 미분하면:

dV/dt = -ΔV₀/τ · e-t/τ
d²V/dt² = ΔV₀/τ² · e-t/τ
두 식을 나누면:
τ = −(dV/dt) / (d²V/dt²)
|t = tc
💡 이것이 혁신적인 이유
✅
ΔV₀, R₀, Rp, Cp 등 어떤 파라미터도 미리 알 필요 없음
✅
측정된 전압의 1차·2차 미분비만으로 τ 즉시 계산
✅
τ 계산 후 OCV = V(tc) − (dV/dt)|tc × τ
✅
모든 SOH, 온도, 화학계에 범용 적용 가능
※ 수치 미분: tc 중심 ±90초 데이터에 최소제곱법 적용 (노이즈 저감)
알고리즘 · Critical Time Criterion
언제 계산을 시작하는가? — tc 결정 기준

τ(t)는 이완 초기에 불안정하다가 점점 수렴합니다. "언제 τ가 충분히 안정화되었는가?"를 자동으로 판단하는 기준이 필요합니다.

📏 매 초마다 계산하는 지표
ΔV(t) = V(t+1) − V(t)
|avg.ΔV| = |최근 60초 평균 ΔV|

전압 변화가 작아질수록 τ가 안정화됨 → OCV 예측 가능

🚦 5가지 기준 조건
기준조건특성
C1 ▴|avg.ΔV| < 0.05 mV가장 빠름, 오차 가능
C2 ◆|avg.ΔV| < 0.04 mV중간
C3 ● ★|avg.ΔV| < 0.03 mV최종 채택 기준
C4 ★|avg.ΔV| < 0.02 mV더 엄격, 역효과 가능
C5 ■|avg.ΔV| < 0.025 mV이상 케이스 전용
⚠️ 이상 케이스 감지: k 파라미터
k = (t₂ - t₃) / (t₁ - t₂)
k ≈ 1.3 → 정상 (이상적 RC) → C3 사용
k > 2 → 이상 → C5로 전환
주로 SOC≈50~55% 평탄 구간에서 k>2 발생
